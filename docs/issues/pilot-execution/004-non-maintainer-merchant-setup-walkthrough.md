# Non-maintainer merchant setup walkthrough

GitHub issue: https://github.com/GiraeffleAeffle/agentcart-tempo-shopbridge/issues/20

## What to build

Run and record a merchant setup walkthrough with someone who is not the repo
maintainer, then turn the findings into concrete setup-doc and checklist fixes.
This slice is HITL because it needs a real person to follow the merchant setup
path and report where it breaks down.

## Acceptance criteria

- [ ] A non-maintainer can start from the merchant setup docs and reach a
  configured staging ShopBridge install.
- [ ] The walkthrough records every place where the operator needed
  repo-maintainer help.
- [ ] Setup docs are updated for the top blockers found in the walkthrough.
- [ ] The pilot evidence folder includes the walkthrough notes or a link to
  them.
- [ ] Remaining merchant UX blockers are captured as follow-up issues with
  severity.
